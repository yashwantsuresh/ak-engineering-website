# AK Engineering Website — VS Code Setup

1. Extract this project folder anywhere on your computer.
2. Open the folder in Visual Studio Code.
3. Open **Terminal** in VS Code.
4. Run `npm install` once.
5. Run `npm run dev`.
6. Open the local URL shown in the terminal (normally `http://localhost:3000`).

## Useful commands

- `npm run dev` — run the website locally
- `npm run build` — create a production build

## Main files

- `app/page.tsx` — website text, products, sections and contact details
- `app/globals.css` — website colors, layout and design
- `public/` — logo and other image files
